# Github issues to hexo

## Installation
Available from `npm`

```shell
npm install github-issues-to-blog -g
```

## Usage

See output of help:

```shell
github-issues-to-hexo --help
```

Or, example:

```shell
cd /path/to/hexo-blog-source/
github-issues-to-hexo -u <github-username> -u <github-repo> -d source/_posts/
```
